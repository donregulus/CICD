----=>Connect Docker agent to jenkins Docker

1-docker exec -it $(jenkins_container_name) curl -sO http://$(jenkins_container_name):8080/jnlpJars/agent.jar
example
docker exec -it ci_cd_jenkins_linux_agent curl -sO http://ci_cd_jenkins:8080/jnlpJars/agent.jar

2-docker exec -it $(jenkins_container_name) java -jar agent.jar -url http://$(jenkins_container_name):8080/ -secret $(node_secret_key) -name "$(node_agent_name)" -webSocket -workDir "/home/jenkins/agent"
example

If you want to stop seeing the "Unsecure" warning right now:
Import the certificate CA (stroalgoCA.crt)used to sign jenkins server certificates used by nginx proxy into your Windows Trusted Root Store.
1. Locate and Prepare the Certificate
    1.1-Open your wsl
    1.2-Copy stroalgoCA.crt.(docker cp nginx_proxy:/stroalgoCA .)
    1.2-Move stroalgoCA.crt to a Windows folder
2. The Import Process
    2.1-Press Win + R, type certlm.msc, and hit Enter. (This opens the Local Machine Certificate Store).
    2.2-On the left sidebar, right-click on the folder named Trusted Root Certification Authorities.
    2.3-Select All Tasks > Import....
    2.4-The Certificate Import Wizard will open. Click Next.
    2.5-Click Browse and select the stroalgoCA.crt file you moved to your Downloads folder. (You may need to change the file type filter to "All Files" to see it). Click Next.
    2.6-Ensure it says "Place all certificates in the following store: Trusted Root Certification Authorities". Click Next, then Finish.
